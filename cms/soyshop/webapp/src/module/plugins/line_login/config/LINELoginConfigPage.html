LINE Loginプラグインの使用方法

LINE Developersにログイン
LINE developersを開き、LINEログインをはじめるを選択します。
サイト用のプロバイダーを作成
画面の指示に従い、プロバイダーを作成します。
Channel IDとChannel Secretを取得
プロバイダーが作成できたら、Channel基本設定を開き、Channel IDとChannel Secretをコピーし、
このページ上部にあるフォームに入力し保存します。
リダイレクト設定を行う
LINE developerの画面でアプリ管理のリンクを探し、リダイレクト設定に下記のURLを登録します。
ログインボタンの動作確認
urlを開き、LINE Loginが表示されていることを確認しましょう
