商品毎パーツモジュール選択読み込みプラグインは商品毎に任意の箇所に出力するパーツモジュールを設定することが出来ます。
商品詳細ページのみで使用可能です。

パーツモジュールに関しては下記の２つの記事をご覧ください。
  • 共通箇所はHTMLモジュールで管理する - saitodev.co
  • 共通箇所をPHPモジュールで管理する - saitodev.co
商品情報の編集画面で表示する項目名

使用方法

・商品詳細ページのテンプレートでblock:id="item"の外側で下記のモジュールを記述します 
<!-- shop:module="common.fixed_form_module" -->
ここに設定した内容を出力します。
<!-- /shop:module="common.fixed_form_module" -->
・商品情報の編集画面でfixed_form_module内で出力するモジュールを選択します。
