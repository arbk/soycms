設定方法

  • カタログマネージャ用XML表示ページ用のテンプレートを用意します。
    表示用テンプレートはフリーページで作成し、作成したテンプレートに<!-- cms:id="facebook_catalog_manager.xml" /-->のみを記述してください。
  • カタログマネージャ用XMLの表示ページを作成します。
    上記で作成したテンプレートを元に、カタログマネージャ用XMLの表示ページを追加する。この時、URLには必ずfacebook_catalog_manager.xmlを入れてください。
  • FacebookページショップやInstagramショッピングで出品したい商品の編集画面を開き、出品にチェックを入れます。
  • Facebook for Businessのカタログマネージャでカタログを作成します。
    カタログマネージャでFacebookカタログを作成する | Facebook Businessヘルプセンター
  • スケジュール設定したフィードを利用でを登録します。
  • 設定方法の詳しい記述はSOY ShopでFacebookページショップと連携する - saitodev.coをご覧ください。
