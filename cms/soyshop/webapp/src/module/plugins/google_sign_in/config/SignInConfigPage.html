Google Sign-In For Websiteプラグインの使用方法

Google Cloud Platformにログイン
Google Cloud Platformを開き、サイドバーにあるAPIとサービスのリンクをクリックします。
OAuth 2.0 クライアント IDを作成
サイドバーにある認証情報のリンクをクリックし、認証情報の作成ボタンからOAuth 2.0クライアント IDを登録します。
承認済みの JavaScript 生成元では
承認済みのリダイレクト URIではを登録します。
OAuth 同意画面でサイトの情報を登録
OAuth 同意画面を開き、画面の指示に従い、サイトの情報を入力します。
クライアントIDを取得
クライアントIDが生成されたら、
このページ上部にあるフォームに入力し保存します。
